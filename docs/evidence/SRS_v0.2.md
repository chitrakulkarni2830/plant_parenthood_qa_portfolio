# 1. Introduction

## 1.1 Purpose

This document defines the functional and non-functional requirements of the Plant Parenthood web application.

The purpose of this application is to help users discover plants, search for plant information, and receive personalized care recommendations through an intuitive and responsive web interface.

This document serves as the baseline for QA planning, test case creation, defect tracking, and future enhancements.

---

## 1.2 Scope

Plant Parenthood is a web-based application that enables users to:

* Login using their name or continue as a guest.
* Browse plant information.
* Search plants using multiple search methods.
* Receive plant care recommendations.
* View plant statistics.
* Access the application across desktop and mobile devices.
