# 🌱 Plant Parenthood

A responsive web application designed to help users discover plants, search plant information, and receive personalized plant care recommendations.

This project is being developed as an end-to-end Software Development Lifecycle (SDLC) portfolio project, showcasing development, quality assurance, version control, and release management practices.

---

## 🚀 Current Version

**Latest Release:** `v1.0.0`

The application has completed its initial development phase and is now entering the Quality Assurance phase.

---

## ✨ Features

* 🌿 Responsive Landing Page
* 🔐 User Login
* 👤 Continue as Guest
* 📊 Interactive Dashboard
* 🔍 Search Plants by:

  * Plant Name
  * Category
  * Sunlight Requirement
  * Water Requirement
* 💧 Personalized Plant Care Recommendations
* 🌱 Plant Count Dashboard
* 📱 Responsive Design
